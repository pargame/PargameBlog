# 정렬 알고리즘

정렬 알고리즘은 데이터를 특정 순서로 배열하는 방법입니다.

## 주요 정렬 알고리즘

- [[버블정렬]]: 인접한 원소를 비교하며 정렬
- [[퀵정렬]]: 분할 정복을 사용한 효율적인 정렬
- [[병합정렬]]: 안정적이고 일관된 성능의 정렬

## 시간 복잡도

각 알고리즘의 [[시간복잡도]]를 비교해보면:

| 알고리즘 | 최선 | 평균 | 최악 |
|----------|------|------|------|
| 버블정렬 | O(n) | O(n²) | O(n²) |
| 퀵정렬   | O(n log n) | O(n log n) | O(n²) |
| 병합정렬 | O(n log n) | O(n log n) | O(n log n) |
